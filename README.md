# JV Pulse Report System

People & Culture Pulse Report build system for Josiah Venture.
Covers Director Review workbook (Excel) and PDF Pulse Report generation.

## Files

- `build_country_pulse_report.py` — builds the Director Review Excel workbook
- `build_pdf_report.py` — builds the PDF Pulse Report via HTML/weasyprint
- `Survey_Basics_Corrections.json` — manual Survey Basics lookup fixes
- `JV_PULSE_REPORT_MASTER_INSTRUCTIONS.md` — complete Claude build instructions
- `CLAUDE_INSTRUCTIONS_Director_Review.md` — Director Review specific instructions
- `SCORING_INSTRUCTIONS.md` — three-factor DIST scoring reference

## Usage

Upload the master instructions + build script + Survey Basics files to a new Claude
chat along with the country raw data file. Claude will handle the rest.
